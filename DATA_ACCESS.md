# Data Access

The full compressed dental panoramic image dataset is intentionally excluded from this
repository because the source NPZ file is approximately 182 MB and exceeds GitHub's
100 MB per-file limit.

Place the dataset locally at:

`data/raw/Dental-Panaromic-Autoencoder.npz`

The expected archive keys are `x_train`, `y_train`, `x_test`, and `y_test`.

The repository does not redistribute patient imagery. Use the dataset only under the
terms and permissions associated with the source from which you obtained it.
